= FRANCAIS : =

<a href="http://www.resaline.net/" target="_blank">Resaline</a>  offre une solution innovante de gestion des disponibilités et de réservation en ligne pour locations de tourisme indépendantes, destinée aux propriétaires de gites, chambres d'hôtes, locations de vacances, hébergements insolites, ...

Le calendrier interactif Resaline s'intègre facilement à votre site, et permet ainsi à l'internaute de :
<ul>
<li>connaitre vos disponibilités</li>
<li>sélectionner les options que vous proposez</li>
<li>afficher le montant de son séjour</li>
<li>éditer le contrat de location automatiquement</li>
<li>voir l'itinéraire pour se rendre sur le lieu de séjour</li>
<li>réserver en ligne</li>
</ul>
Le calendrier et le contrat de location sont traduisibles en plusieurs langues pour faciliter la communication avec les clients étrangers.

IMPORTANT: Le plugin nécessite l'existence d'un compte actif sur <a href="http://www.resaline.net/" target="_blank">Resaline.net</a> pour lier votre calendrier à votre site wordpress.
<strong>C'est totalement gratuit.</strong>
= ENGLISH : =

<a href="http://www.resaline.net/">Resaline</a> offers an inovative booking calendar online reservation system for individuals ( Bed & Breakfast, cottages, holidays rentals ... )

<ul>

<li>choose any options you're proposing</li>

== Installation ==
<a href="http://www.resaline.fr/static/tuto/sitewordpress/resaline_wpdoc.pdf" target="_blank"><strong>Cliquez ici pour le tutoriel d'installation en pdf</strong></a>

= FRANCAIS : =

1. Créez votre compte sur <a href="http://www.resaline.net/" target="_blank">resaline.net</a>
2. Paramétrez votre établissement et les tarifs sur www.Resaline.net    
3. Déposez le plugin sur le répertoire /wp-content/plugins/
4. Activez le plugin dans le menu 'Plugins' de WordPress
5. Entrez votre nom d'utilisateur dans le paramétrage du plugin afin de récupérer vos calendriers publiés et paramétrés sur resaline.net
6. Insérez le calendrier correspondant à votre établissement sur votre page grâce au bouton d'insertion intégré à votre éditeur de Posts.
7. Par la suite vous gérerez vos réservations directement sur www.Resaline.net

= ENGLISH : =

1. Create your account in <a href="http://www.resaline.net/" target="_blank">resaline.net</a>
2. Configure your establishment and price lists(rates) in <a href="http://www.resaline.net/ target="_blank"">resaline.net</a>
3. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
4. Activate the plugin through the 'Plugins' menu in WordPress
5. Enter your username in the resaline plugins configuration to link and update your booking calendars
6. Integrate your booking calendar thanks to the integrated button in the post editor
7. Next you  will manage your reservations directly in <a href="http://www.resaline.net/" target="_blank">resaline.net</a>

== Frequently asked questions ==

= What are the features of calendar ? =
1. You could define your own colors for each tariffs periods of the calendar.
2. Booking Calendar calculates many properties in one time.
3. Booking Calendar calculates tariffs in Week, Day, MidWeek, and Week End.
4. Booking Calendar calculates tariffs with several options. 
5. Booking Calendar prints rental agreement with your own specificities
6. Booking Calendar calculates tariffs with supplements and reductions. 
7. Visitor could see in one time many tariffs on Calendar
8. Booking Calendar could be render on Facebook page.
9. Booking Calendar could be render on many platforms sites.
10. Booking Calendar can display tariffs directly in calendar.
11. Booking Calendar can be use in simple mode(without displaying tariffs in calendar).  

= Booking in calendar ? =
1. Visitor can directly book from the calendar.
2. Booking Calendar is only one screen and so what writes visitor to book is memorized.
3. Booking system make alerts to customers and owners.
4. You can reedit rental agreement
5. You can block the calendar after according a booking.  

= Where could I set appearance of calendar ? =
1. You can set the booking calendar's appearance directly in www.resaline.net. You can set the calendar as a simple calendar or you could set calendar with tariffs in each weeks. 
2. You can set tariffs' periods appearance in www.resaline.net. You can set the periods'number you want, there are no limits with Resaline Booking Calendar 

= Booking Calendar display ? =
1. Booking Calendar could be integrate as iframe, so calendar could be display on several sites with the same informations.
2. Booking Calendar could be display also in more several platforms and lots of CMS or WebSite helps sites.
