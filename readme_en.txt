== Description ==

<a href="http://www.resaline.net/">Resaline</a> offers an inovative booking online reservation system for individuals ( Bed & Breakfast, cottages, holidays rentals ... )

The interactive calendar fits to any website and let the user:
- be aware of your availabilities
- choose any options you're proposing
- edit the rental agreement
- display the itinerary
- book online!

The rental agreement and calendar are both multi-languages translatable to facilitate communication with foreign customer.

IMPORTANT: You need an active account in <a href="http://www.resaline.net/">resaline.net</a> to link your Resaline calendar to your wordpress website.

== Installation ==

1. Create your account in <a href="http://www.resaline.net/">resaline.net</a>
2. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enter your account number in the resaline plugins configuration
5. Integrate your booking calendar thanks to the integrated button in the post editor
